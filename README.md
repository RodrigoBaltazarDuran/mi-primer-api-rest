# mi-primer-api-rest

Este proyecto esta creado con fines de práctica, para el aprendizaje y conocimiento de las API REST, dentro de este proyecto se creo un pequeño CRUD con las siguientes tecnologías

- Java 17
- Spring Boot
- Swagger
